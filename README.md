# Road Accident Analysis Dashboard (Excel)

## Project Overview

This project presents an interactive Road Accident Analysis Dashboard built using Microsoft Excel. The dashboard analyzes accident data to understand casualty distribution, accident trends, and risk factors related to road type, location, and vehicle type.

The objective of this project is to transform raw accident data into meaningful business insights through data visualization.

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* Slicers
* Data Visualization

## Key Metrics

The dashboard highlights several important accident statistics:

* Total Casualties
* Fatal Casualties
* Serious Casualties
* Slight Casualties
* Casualties by Vehicle Type

---

## Key Insights

* Most casualties occur in **car-related accidents** compared to other vehicle types.
* **Urban areas show higher accident frequency** than rural areas.
* **Single carriageway roads account for the highest number of accidents.**
* Casualties tend to vary across different months, indicating possible seasonal trends.
* Road surface conditions such as **wet or snowy roads increase accident risks.**

---

## Dashboard Features

* Interactive filters for **Year and Urban/Rural area**
* Casualty analysis by **vehicle type**
* Monthly accident trend comparison
* Casualty distribution by **road type and road surface**
* Urban vs rural accident comparison
* Light condition analysis (Daylight vs Darkness)
